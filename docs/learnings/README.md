# 학습 문서 (learnings)

이 폴더는 finance_agent 코드베이스를 처음 보는 사람이 순서대로 읽으며 시스템을 이해하도록 쓴 학습 문서 묶음이다. 각 문서는 코드의 특정 영역을 맡고, 실제 파일·함수·테이블로 역추적할 수 있게 쓴다.

## 읽는 순서

00부터 09까지 번호 순서로 읽는 것을 권장한다. 각 문서 끝의 "다음에 읽을 문서" 링크를 따라가면 같은 순서가 된다. 특정 영역만 필요하면 아래 표에서 골라 읽어도 된다 — 단 00(전체 개요)은 먼저 읽는 것이 좋다.

## 문서 공통 규약

모든 문서는 같은 뼈대를 따른다.

- **한 줄 요약** → **비개발자 설명**: 개발 지식 없이 읽는 진입부
- **설계도** + **다이어그램 코드 매핑**: mermaid 그림의 각 박스가 어느 코드인지 표로 연결
- **코드/폴더 매핑**: 관련 파일과 역할
- **왜 이렇게 만들었나**: 설계 근거 — 운영 중 실측으로 배운 교훈 포함
- **관련 테스트**: 각 테스트가 막는 사고
- **다음에 읽을 문서**: 권장 순서의 다음 문서

도메인 주의: 이 시스템의 종목 언급은 투자 권유가 아니라 "수집된 뉴스/공시 근거 기준 영향도 분석"이다. 문서도 같은 관점으로 쓴다.
